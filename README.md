# jp-av

FAN◯AからAV女優一覧をスクレイピングで抜き出してyamlファイルに出力するスクリプト

#### 出力したyamlファイルは以下
[dist/actress.yaml](https://raw.githubusercontent.com/genkaieng/jp-av/refs/heads/main/dist/actress.yaml)

## スクリプトを実行して最新を出力する場合

必要なパッケージをインストール

```sh
pip install -r requirements.txt
```

スクリプト実行

```sh
python src/main.py
```
